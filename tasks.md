# مهام تطوير نظام العقارات

## نظرة عامة

هذا الملف يحتوي على المهام المطلوبة لتطوير نظام إدارة العقارات بناءً على الملاحظات المقدمة.

---

## 1. عقارات الوسطاء (Broker Properties)

### 1.1 نظام تصنيف العقارات

- [ ] إنشاء 4 تصنيفات للعقارات:
  - **جديد** - عقارات تم إضافتها حديثاً
  - **تحت التسويق** - عقارات يتم التسويق لها حالياً
  - **تمت الصفقة** - عقارات تم بيعها/تأجيرها
  - **محذوف** - عقارات تم حذفها
- [ ] إضافة فلاتر للتنقل بين التصنيفات
- [ ] عرض العدد الإجمالي لكل تصنيف

### 1.2 زر "قم بالتسويق للعقار"

- [ ] إضافة زر "قم بالتسويق للعقار" (يظهر للوسطاء فقط)
- [ ] عند الضغط على الزر → إظهار نافذة تأكيد (صورة 2)
- [ ] عند الضغط على "موافقة" → التحقق من الاشتراك في الباقة
- [ ] إذا لم يكن مشترك → إظهار صفحة الباقات (صورة 4)
- [ ] ربط النظام بنظام الباقات للتحقق من الاشتراك

### 1.3 التحكم في الصلاحيات

- [ ] التأكد من أن خيار التسويق يظهر للوسطاء فقط
- [ ] إخفاء الخيار عن المستخدمين العاديين والملاك

---

## 2. عقارات الملاك (Owner Properties)

### 2.1 واجهة إضافة العقار

- [ ] إضافة زر "إضافة عقار كمالك" في لوحة تحكم المالك
- [ ] إنشاء نموذج إضافة عقار خاص بالملاك
- [ ] التفريق بين عقارات المالك وعقارات الوسيط

---

## 3. عقارات المطورين (Developer Properties)

### 3.1 قسم المطورين

- [ ] إنشاء قسم خاص بعقارات المطورين
- [ ] تصميم واجهة عرض مشاريع المطورين
- [ ] إضافة تفاصيل المشاريع والوحدات المتاحة

---

## 4. صفحة الوسطاء / العقاريين (Agents Page)

### 4.1 واجهة صفحة الوسيط

- [ ] إضافة زرارين في صفحة الوسيط:
  - **اطلب عقار** - للمستخدم الذي يبحث عن عقار
  - **اعرض عقارك** - للمستخدم الذي يريد عرض عقاره
- [ ] تصميم النوافذ المنبثقة لكل زر
- [ ] ربط الأزرار بالوظائف المناسبة

---

## 5. نظام الإعلانات (Advertisements System)

### 5.1 إضافة إعلان جديد - الخطوة الأولى

- [ ] إنشاء زر "إضافة إعلان"
- [ ] تصميم النموذج الأول (صورة 5)
- [ ] إضافة زر "التالي" للانتقال للخطوة التالية

### 5.2 التحقق من الرخصة - الخطوة الثانية

- [ ] إضافة اختيارين:
  - **لديه رخصة** → الإعلان ينزل فوراً
  - **ليس لديه رخصة** → إظهار نموذج إضافي (صورة 7)
- [ ] التحقق من صحة الرخصة إذا تم إدخالها

### 5.3 تفاصيل العقار - الخطوة الثالثة

- [ ] **أنواع العقارات** (Select) - صورة 8
  - شقة، فيلا، أرض، مبنى تجاري، إلخ
- [ ] **نوع الإعلان** - صورة 9
  - بيع / إيجار
- [ ] **تفاصيل الإيجار** (إذا كان إيجار) - صورة 10
  - شهري، سنوي، إلخ
- [ ] **نوع السكن** - صورة 11
  - عائلي، عزاب، إلخ
- [ ] **المدينة** - صورة 12
  - قائمة المدن
- [ ] **الحي** - صورة 13
  - قائمة الأحياء (حسب المدينة المختارة)

### 5.4 تفاصيل إضافية

- [ ] **المساحة** - حقل نصي/رقمي
- [ ] **السعر الإجمالي** - حقل رقمي
- [ ] **سعر المتر** - حقل رقمي (يمكن حسابه تلقائياً)
- [ ] **استخدامات العقار** - صورة 14
  - سكني، تجاري، صناعي، إلخ
- [ ] **خدمات العقار** - صورة 15
  - مواقف، مصعد، حراسة، إلخ
- [ ] **عرض الشارع** - حقل نصي
- [ ] **واجهة الشارع** - صورة 16
  - شمال، جنوب، شرق، غرب
- [ ] **الالتزامات** - حقل نصي (اختياري)
- [ ] **التفاصيل** - حقل نصي طويل
- [ ] **الصور والفيديوهات** - رفع ملفات متعددة

### 5.5 طرق التواصل

- [ ] **أنواع التواصل** - صورة 17
  - هاتف، واتساب، إيميل، إلخ
- [ ] تحديد طرق التواصل المفضلة

### 5.6 معاينة ونشر الإعلان

- [ ] صفحة معاينة الإعلان قبل النشر
  - عرض بيانات الإعلان كاملة
  - عرض بيانات الوسيط/المعلن
- [ ] زر "نشر الإعلان"

### 5.7 التسجيل في الهيئة العامة للعقار

- [ ] **رخصة الهيئة** - صورة 18
  - حقل إدخال رقم الرخصة
  - التحقق من الرخصة
- [ ] **هوية المعلن** - صورة 18
  - رفع صورة الهوية
  - التحقق من الهوية

### 5.8 خيار التعاقد مع وسيط

- [ ] زر "أرغب بالتعاقد مع وسيط عقاري"
- [ ] عند الضغط → إظهار قائمة الوسطاء (صورة 19)
- [ ] إمكانية اختيار وسيط للتعاقد معه
